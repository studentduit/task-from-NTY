# C++ Concurrency Tasks

## Description
This project demonstrates basic concurrency concepts in C++:

- Race condition
- Mutex (lock)
- Parallel execution
- Async / Future
- Pipeline processing

## Tasks
1. Race condition
2. Mutex fix
3. Functional programming
4. Parallel map
5. Performance comparison
6. CPU-bound tasks
7. Pipeline
8. Safe execution
9. API simulation

## Build
g++ -std=c++17 -pthread main.cpp -o app

## Run
./app
